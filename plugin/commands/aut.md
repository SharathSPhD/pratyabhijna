Delegate to the `pratyabhijna-divergent-thinker` agent. The agent must:

* Run `iccha` with `K = --K` (default 8).
* Score with `apohana` + `ananda` and rank with `jnana`.
* Report fluency, originality, elaboration, flexibility per AUT.

Return all K candidates ranked by posterior with the BMR winner highlighted.
